Class Dropio_Api

Description
Description | Vars (details) | Methods (details) Constants (details)

Dropio_Api is a client for the Dropio API and the basis for all the other helper classes.

This can be used to access all the functionality of the API.

If an error is returned from the API, a Dropio_Api_Exception is thrown.

Example to get details on a drop.

try { $api = new Dropio_Api(API_KEY); $response = $api->request('GET', '/drops/php_api_lib'); print_r($response); } catch (Dropio_Api_Exception $e) { die("Error:" . $e->getMessage()); }
