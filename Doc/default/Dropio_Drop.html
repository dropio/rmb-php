Class Dropio_Drop

Description
Description | Vars (details) | Methods (details)

Dropio_Drop is used to access all functionality related to a Drop. Most methods are chainable, allowing for the the creation of a drop to happen inline.

For example, to create a drop and to access it.

$drop = Dropio_Drop::instance('dropname')->save();

To load a pre-existing drop and load:

$drop = Dropio_Drop::load('dropname');
